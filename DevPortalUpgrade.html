Upgraded ITA Developer Portal

On October 2, 2019, we launched an upgraded ITA Developer Portal. This upgrade will provide increased platform stability and improved security, providing you, our customers, with a more robust API platform.

What this Means for You

Accessing our APIs via our new API Store will require all users to request an access token. All the legacy API keys will continue to work until close of business January 30, 2020, when the current system will be turned off. During the next four months, please explore our new API Store and sign-up for a new access token. The new access tokens are free and will continue to provide you with access to all our APIs.

Along with the new access tokens, we have also established new user account management policies. For security reasons, we will now be deactivating, then deleting user accounts that do not have any activity associated with them. We have defined user activity as either a login to the API store or the use of an access token associated with the account. The following policies are in place for user accounts and access token:

  • Deactivate after 45 days of inactivity and
  • Delete after 60 days of inactivity.

With the implementation of the API store, we have added throttling to all our available APIs. This will increase the stability of the system and elevate the level of service that we are able to provide to you. The throttling level for each API is available within the documentation for that API.

These changes have been implemented so that we can provide you with a more stable and secure API platform. If you have any questions or concerns about obtaining a new access token, the user account management policies, or our new throttling level, please reach out to us at DataServices@trade.gov.

Subscribing to an API

  1. Sign In using the link at the top right corner of the API store. If you have not done this before, you will first need to create an account.
  2. Please go to the Applications page and generate an access token.
  3. All our APIs are available. Please go to the APIs page and click on an API to view its details.
  4. The selected API’s information will be visible.
  5. Select a service tier (currently we only have one tier) then click on the Subscribe button. If the subscription is successful, a message appears congratulating you on your successful subscription.

Questions?

If you have questions now about the change or think of questions while perusing our new site, please reach out to us at DataServices@trade.gov. We are happy to answer any questions that you have.
